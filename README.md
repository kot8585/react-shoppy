## 프로젝트 상세
- 상품 무한 스크롤 기능 (혼자서 추가한 기능)
- 상품 상세 보기 
- 로그인되어있을 시 장바구니에 상품 추가
- 장바구니 조회
- 관리자일 시 상품 추가 기능
- 장바구니에 담긴 상품 개수 표시
- 로그인 / 로그아웃
